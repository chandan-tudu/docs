Navigating Back to the Previous Page Using JavaScript

JavaScript provides a simple and effective way to achieve this using the window.history object.

In this article, we'll explore how to use JavaScript to go back to the previous page in the browser's history.

Using "window.history"

The window.history object in JavaScript represents the browser's session history.

By utilizing the go method with a negative value, you can instruct the browser to navigate backward in the history.

Let's create a simple function to handle the navigation:

// Function to go back to the previous page
function goBack() {
  // Navigate back in the history
  window.history.go(-1);
}

In this code:

  • The goBack function uses the window.history.go method with a parameter of -1 to navigate back one step in the browser's history.

In this example:

  • A button is included in the HTML with an onclick attribute calling the goBack function when clicked.

  • The goBack function navigates back to the previous page when the button is clicked.
