JavaScript Window.location Cheatsheet

The window.location object in JavaScript provides valuable information about the current URL and allows developers to manipulate it to navigate the browser or extract specific details.

This cheatsheet serves as a quick reference guide for working with window.location in various scenarios.

1. Accessing Components of the URL:

Getting the Full URL:

const fullURL = window.location.href;
console.log(fullURL);

Getting the Host (domain and port):

const host = window.location.host;
console.log(host);

Getting the Domain:

const domain = window.location.hostname;
console.log(domain);

Getting the Path:

const path = window.location.pathname;
console.log(path);

Getting the Protocol:

const protocol = window.location.protocol;
console.log(protocol);

Getting Query Parameters:

const queryParams = window.location.search;
console.log(queryParams);

2. Navigating the Browser:

Redirecting to a New URL:

window.location.href = 'https://example.com';

Reloading the Page:

window.location.reload();

Forward and Backward Navigation:

// Go back one page in the browser history
window.history.back();

// Go forward one page in the browser history
window.history.forward();

3. Modifying the URL:

Changing the Hash (Fragment Identifier):

window.location.hash = '#section1';

Modifying Query Parameters:

const newParams = '?page=2&category=javascript';
window.location.search = newParams;

Changing the Path:

window.location.pathname = '/newPath';

4. Extracting URL Information:

Parsing the Query Parameters:

const params = new URLSearchParams(window.location.search);
console.log(params.get('paramName'));

Checking if the URL contains a specific string:

const containsString = window.location.href.includes('example');
console.log(containsString);

Extracting Fragments:

const fragment = window.location.hash.substring(1);
console.log(fragment);

5. Working with Cookies:

Redirecting Based on Cookie:

const userLoggedIn = document.cookie.includes('loggedIn=true');

if (!userLoggedIn) {
    window.location.href = '/login';
}

Conclusion:

The window.location object in JavaScript is a powerful tool for manipulating and navigating the browser's URL.

This cheatsheet provides a quick reference for common scenarios, whether you need to extract specific components of the URL, navigate to a new page, or modify the URL dynamically.

Understanding these techniques empowers developers to create more dynamic and interactive web applications.
