How to Truncate a String in JavaScript?

Truncating a string in JavaScript means reducing its length to a specified maximum number of characters while preserving its meaning and readability.

This is a common operation when displaying text in user interfaces, blog excerpts, or generating summaries.

In this article, we'll explore different methods and techniques to truncate a string in JavaScript.

Using String Slice

The slice() method in JavaScript allows you to extract a portion of a string starting from a specified index. To truncate a string to a certain length, you can use slice() along with the desired length as the endpoint.

Here's an example:

function truncateString(str, maxLength) {
  if (str.length > maxLength) {
    return str.slice(0, maxLength) + "...";
  }
  return str;
}

const originalString = "Lorem ipsum dolor sit amet, consectetur adipiscing elit.";
const maxLength = 30;

const truncatedString = truncateString(originalString, maxLength);

console.log(truncatedString); // Output: "Lorem ipsum dolor sit amet..."

In this example, the truncateString function takes an input string str and a maxLength. If the length of the input string exceeds the maxLength, it uses slice() to truncate the string to the desired length and appends ellipsis ("...") to indicate that the string has been truncated.

Using String Substr

The substr() method is another way to truncate a string by specifying the starting index and the length of the substring you want to extract.

Here's an example:

function truncateString(str, maxLength) {
  if (str.length > maxLength) {
    return str.substr(0, maxLength) + "...";
  }
  return str;
}

const originalString = "Lorem ipsum dolor sit amet, consectetur adipiscing elit.";
const maxLength = 30;

const truncatedString = truncateString(originalString, maxLength);

console.log(truncatedString); // Output: "Lorem ipsum dolor sit amet..."

In this example, the truncateString function is similar to the previous example but uses the substr() method instead of slice() to achieve the same result.

Using String Substring

The substring() method is similar to slice() but accepts two indices as arguments: the start index and the end index (not the length). To truncate a string to a certain length using substring(), you need to calculate the end index based on the desired length.

Here's an example:

function truncateString(str, maxLength) {
  if (str.length > maxLength) {
    return str.substring(0, maxLength) + "...";
  }
  return str;
}

const originalString = "Lorem ipsum dolor sit amet, consectetur adipiscing elit.";
const maxLength = 30;

const truncatedString = truncateString(originalString, maxLength);

console.log(truncatedString); // Output: "Lorem ipsum dolor sit amet..."

In this example, the truncateString function uses substring() to achieve the same truncation effect as in the previous examples.

Using Regular Expressions

Regular expressions can be a powerful tool for truncating a string based on specific patterns or conditions. In this example, we'll use a regular expression to truncate a string while ensuring that words are not cut in the middle.

function truncateString(str, maxLength) {
  if (str.length > maxLength) {
    const truncated = str.substr(0, maxLength);
    return truncated.replace(/\s+\S*$/, '...');
  }
  return str;
}

const originalString = "Lorem ipsum dolor sit amet, consectetur adipiscing elit.";
const maxLength = 30;

const truncatedString = truncateString(originalString, maxLength);

console.log(truncatedString); // Output: "Lorem ipsum dolor sit amet..."

In this example, the truncateString function first truncates the string to the desired length using substr(). Then, it uses a regular expression (/\s+\S*$/) to find the last space character followed by non-space characters at the end of the truncated string and replaces it with ellipsis ("..."). This ensures that words are not cut in the middle while truncating the string.

Conclusion

Truncating a string in JavaScript is a common operation when dealing with text data in web development. Whether you choose to use slice(), substr(), substring(), or regular expressions, the method you use depends on your specific requirements and coding style. Keep in mind that when truncating strings, it's essential to provide meaningful and readable summaries while preserving the overall context and content of the text.
