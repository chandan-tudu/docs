How to Remove All Falsy Values from an Array in JavaScript?

In JavaScript, you often encounter arrays that contain a mix of truthy and falsy values. Falsy values include undefined, null, false, 0, NaN, and an empty string ("").

Removing falsy values from an array is a common operation when you want to clean up your data.

In this article, we'll explore different techniques to remove all falsy values from an array in JavaScript.

Using the filter() Method

The filter() method is a built-in array method in JavaScript that creates a new array containing all elements that pass a provided test. To remove falsy values from an array, you can use the filter() method with a callback function that checks if each element is truthy.

Here's an example:

const array = [1, 0, "hello", "", false, true, null, undefined, NaN];

const filteredArray = array.filter((element) => !!element);

console.log(filteredArray);
// Output: [1, "hello", true]

In this example, we use the filter() method with a callback function that checks if each element is truthy by converting it to a boolean using the double negation operator (!!). The resulting filteredArray contains only the truthy values from the original array.

Using a for...of Loop

You can also remove falsy values from an array by iterating through it using a for...of loop and adding the truthy elements to a new array.

Here's an example:

const array = [1, 0, "hello", "", false, true, null, undefined, NaN];
const filteredArray = [];

for (const element of array) {
  if (element) {
    filteredArray.push(element);
  }
}

console.log(filteredArray);
// Output: [1, "hello", true]

In this example, we initialize an empty array called filteredArray and use a for...of loop to iterate through the original array. We add elements to filteredArray only if they are truthy, effectively removing falsy values.

Using the reduce() Method

The reduce() method can also be used to remove falsy values from an array. You can use reduce() to accumulate the truthy elements into a new array.

Here's an example:

const array = [1, 0, "hello", "", false, true, null, undefined, NaN];

const filteredArray = array.reduce((acc, element) => {
  if (element) {
    acc.push(element);
  }
  return acc;
}, []);

console.log(filteredArray);
// Output: [1, "hello", true]

In this example, we use the reduce() method to accumulate truthy elements into the filteredArray. The initial value for the accumulator (acc) is an empty array. We check if each element is truthy, and if it is, we push it onto the accumulator.

Using the Boolean Constructor

Another concise way to remove falsy values from an array is by using the Boolean constructor function as the filtering criteria. The Boolean constructor returns true for truthy values and false for falsy values. When used with the filter() method, it effectively filters out all falsy elements.

Here's an example:

const array = [1, 0, "hello", "", false, true, null, undefined, NaN];

const filteredArray = array.filter(Boolean);

console.log(filteredArray);
// Output: [1, "hello", true]

In this example, we use the filter() method with the Boolean constructor as the callback function. This method will remove all falsy elements from the original array.

Conclusion

Removing falsy values from an array in JavaScript is a common task when working with data. Depending on your preference and coding style, you can choose one of the methods described in this article. Whether you use filter(), a for...of loop, reduce(), or the Boolean constructor, each approach provides an effective way to clean up your array and keep only the truthy values.
