How to Check if a Number is a Float or Integer in JavaScript?

In JavaScript, numbers can be categorized into two main types: integers and floating-point numbers (floats).

Integers are whole numbers without a decimal point, while floating-point numbers have decimal parts.

There are several ways to determine whether a number is a float or an integer in JavaScript, and in this comprehensive guide, we will explore various methods and techniques to accomplish this task.

Using the Modulus Operator (%)

One of the simplest and most commonly used methods to check if a number is an integer is by using the modulus operator (%). The modulus operator calculates the remainder when one number is divided by another. If the remainder is 0, the number is an integer; otherwise, it's a float.

function isInteger(number) {
  return number % 1 === 0;
}

console.log(isInteger(5));      // Output: true
console.log(isInteger(5.5));    // Output: false
console.log(isInteger(-10));    // Output: true
console.log(isInteger(0.25));   // Output: false

In the isInteger function, we calculate the remainder when number is divided by 1. If the remainder is 0, the number is considered an integer.

Using the Number.isInteger() Method

JavaScript provides the Number.isInteger() method, which is a more explicit way to check if a value is an integer. It returns true if the value is an integer and false otherwise.

console.log(Number.isInteger(5));      // Output: true
console.log(Number.isInteger(5.5));    // Output: false
console.log(Number.isInteger(-10));    // Output: true
console.log(Number.isInteger(0.25));   // Output: false

The Number.isInteger() method directly checks if the value passed to it is an integer, providing a clear and concise way to perform the check.

Using parseInt() and parseFloat()

Another approach to distinguishing between floats and integers is by using the parseInt() and parseFloat() functions to parse a number as an integer or a float, respectively. You can then compare the parsed result with the original number to determine its type.

function isInteger(number) {
  return parseInt(number) === number;
}

console.log(isInteger(5));      // Output: true
console.log(isInteger(5.5));    // Output: false
console.log(isInteger(-10));    // Output: true
console.log(isInteger(0.25));   // Output: false

In this approach, we parse the number using parseInt() and compare it to the original number. If they match, the number is an integer.

Conclusion

Determining whether a number is a float or an integer in JavaScript can be achieved using various methods, including the modulus operator, the Number.isInteger() method, and parsing functions like parseInt() and parseFloat(). Depending on your specific use case and coding style, you can choose the method that best suits your needs. These techniques are essential when working with numeric data in JavaScript and can help ensure that your code behaves correctly based on the type of number you are dealing with.
