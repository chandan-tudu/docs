Stripping HTML Tags from Text Using Plain JavaScript

When working with HTML content, there are situations where you need to remove HTML tags from a given text. This process is often referred to as "stripping" HTML tags.

Whether you want to extract plain text for displaying content or perform data processing, stripping HTML tags using plain JavaScript is a useful skill.

In this comprehensive guide, we'll explore various techniques to accomplish this task.

Prerequisites

Before we proceed, make sure you have a basic understanding of HTML, JavaScript, and text manipulation. You should also have a code editor and a web browser for testing your code.

Method 1: Using a Regular Expression

One of the most common and efficient methods for removing HTML tags is by using a regular expression. Here's how you can do it:

function stripHTMLTags(input) {
    return input.replace(/<[^>]*>/g, "");
}

const htmlContent = "<p>This is <strong>HTML</strong> content.</p>";
const plainText = stripHTMLTags(htmlContent);

console.log(plainText); // Output: "This is HTML content."

In this code:

  • The stripHTMLTags function takes an input string and uses the replace method with a regular expression (/<[^>]*>/g) to match and replace all HTML tags with an empty string.

Method 2: Using a Temporary Element

Another approach is to create a temporary HTML element and set its innerHTML property to the input HTML content. Then, you can extract the text content of the element:

function stripHTMLTags(input) {
    const tempElement = document.createElement("div");
    tempElement.innerHTML = input;
    return tempElement.textContent || tempElement.innerText || "";
}

const htmlContent = "<p>This is <strong>HTML</strong> content.</p>";
const plainText = stripHTMLTags(htmlContent);

console.log(plainText); // Output: "This is HTML content."

In this code:

  • We create a temporary div element using document.createElement.
  • We set the innerHTML property of the element to the input HTML content.
  • We retrieve the text content of the element using textContent or, if not supported, innerText.

Method 3: Using DOMParser

The DOMParser API provides a method to parse HTML content as a DOM tree. You can use it to remove HTML tags and extract plain text:

function stripHTMLTags(input) {
    const parser = new DOMParser();
    const doc = parser.parseFromString(input, "text/html");
    return doc.body.textContent || "";
}

const htmlContent = "<p>This is <strong>HTML</strong> content.</p>";
const plainText = stripHTMLTags(htmlContent);

console.log(plainText); // Output: "This is HTML content."

In this code:

  • We create a DOMParser instance and use it to parse the input HTML content.
  • We access the textContent property of the body element of the parsed document to get the plain text.

Method 4: Using innerText (Limited Support)

For browsers that support the innerText property, you can directly use it to get the text content of an element, effectively stripping HTML tags:

function stripHTMLTags(input) {
    const tempElement = document.createElement("div");
    tempElement.innerHTML = input;
    return tempElement.innerText;
}

const htmlContent = "<p>This is <strong>HTML</strong> content.</p>";
const plainText = stripHTMLTags(htmlContent);

console.log(plainText); // Output: "This is HTML content."

Keep in mind that the innerText property may not be supported in all browsers, so it's less reliable than the previous methods.

Conclusion

Stripping HTML tags from text using plain JavaScript is a valuable skill when you need to work with HTML content in a text-centric context. You can choose the method that best fits your project's requirements and browser compatibility. Whether you prefer regular expressions, temporary elements, the DOMParser API, or innerText, these techniques will help you efficiently extract plain text from HTML content.
