How to Calculate Percentage between Two Numbers in JavaScript?

Calculating percentages is a common task in many JavaScript applications, especially when dealing with data analysis, statistics, or creating dynamic web pages.

In this article, we'll explore different methods for calculating the percentage between two numbers in JavaScript, along with practical examples.

Understanding Percentage Calculation

Before we dive into the methods, let's clarify how percentage calculation works. In mathematics, a percentage is a ratio or fraction expressed as a portion of 100. To calculate the percentage between two numbers, you typically follow this formula:

Percentage = (Part / Whole) * 100
  • Part: The smaller value or portion you want to find the percentage of.
  • Whole: The larger value or the total reference point.

Method 1: Using Simple Division

The most straightforward way to calculate the percentage between two numbers is by performing simple division and then multiplying by 100. This method works when you want to find the percentage of one number relative to another.

const part = 25;
const whole = 100;

const percentage = (part / whole) * 100;

In this example, percentage will be 25% because 25 is 25% of 100.

Method 2: Using a Custom Function

You can create a custom JavaScript function to calculate percentages to make your code more reusable and readable. Here's an example of a calculatePercentage function:

function calculatePercentage(part, whole) {
  return (part / whole) * 100;
}

const part = 25;
const whole = 100;

const percentage = calculatePercentage(part, whole);

This approach is particularly useful when you need to calculate percentages multiple times in your code.

Method 3: Using Template Literals for Formatting

When displaying calculated percentages, you can use template literals to format the output with a specified number of decimal places. This is helpful for presenting percentages with precision.

const part = 25;
const whole = 100;

const percentage = (part / whole) * 100;

const formattedPercentage = `${percentage.toFixed(2)}%`;

In this example, formattedPercentage will be "25.00%", displaying the percentage with two decimal places.

Method 4: Using a Predefined Function

JavaScript provides the Intl.NumberFormat object, which allows you to format numbers, including percentages, with localization support. This method is especially useful when you need to display percentages in different locales.

const part = 25;
const whole = 100;

const percentage = (part / whole) * 100;

const formatter = new Intl.NumberFormat('en-US', {
  style: 'percent',
  minimumFractionDigits: 2,
});

const formattedPercentage = formatter.format(percentage);

Here, formattedPercentage will be "25.00%". You can adjust the locale and formatting options to suit your needs.

Conclusion

Calculating percentages between two numbers is a fundamental operation in various JavaScript applications, from data analysis to creating dynamic user interfaces. Depending on your specific requirements and the context of your code, you can choose from these methods to calculate and format percentages effectively. Whether it's simple division, custom functions, template literals for formatting, or using the Intl.NumberFormat object for localization support, JavaScript provides the tools to handle percentage calculations with precision and flexibility.
