How to Send Request Query Parameters with Axios

When working with web services and APIs in JavaScript, you often need to send HTTP requests with query parameters to fetch or filter data.

Axios is a popular JavaScript library for making HTTP requests, and it provides a convenient way to include query parameters in your requests.

In this comprehensive guide, we will explore how to send request query parameters with Axios.

1. Introduction to Axios

Axios is a promise-based HTTP client for the browser and Node.js. It provides an easy-to-use API for making HTTP requests to fetch data from external sources or interact with web services and APIs.

To get started, you need to include Axios in your project. You can either download it manually or use a package manager like npm or yarn:

npm install axios

Now, let's see how to use Axios to send HTTP requests with query parameters.

2. Sending GET Requests with Query Parameters

Method 1: Using a Query String

You can send query parameters in the URL by manually constructing the query string. Axios allows you to include query parameters directly in the URL.

Here's an example:

const axios = require('axios'); // Import Axios

const baseURL = 'https://api.example.com/data'; // The base URL

// Define query parameters
const params = {
    category: 'books',
    author: 'John Doe',
    sortBy: 'date',
};

// Create the query string from the parameters
const queryString = new URLSearchParams(params).toString();

// Combine the base URL and the query string
const requestURL = `${baseURL}?${queryString}`;

// Send the GET request with the query parameters
axios
    .get(requestURL)
    .then((response) => {
        console.log(response.data);
    })
    .catch((error) => {
        console.error(error);
    });

In this example, we define the query parameters in the params object and use URLSearchParams to create a query string. We then combine the base URL and query string to form the complete request URL.

Method 2: Using the "params" Option

Axios provides a more straightforward way to send query parameters using the params option.

Here's how to do it:

const axios = require('axios'); // Import Axios

const baseURL = 'https://api.example.com/data'; // The base URL

// Define query parameters
const params = {
    category: 'books',
    author: 'John Doe',
    sortBy: 'date',
};

// Send the GET request with the query parameters using the `params` option
axios
    .get(baseURL, { params })
    .then((response) => {
        console.log(response.data);
    })
    .catch((error) => {
        console.error(error);
    });

In this code, we pass the params object directly to the Axios request as an option. Axios will automatically create the query string for you and append it to the URL.

3. Handling Dynamic Query Parameters

To handle dynamic query parameters in your requests, you can build the params object programmatically based on user input or other factors. This allows you to create flexible and customizable requests.

Here's an example of how to handle dynamic query parameters:

const axios = require('axios'); // Import Axios

const baseURL = 'https://api.example.com/data'; // The base URL

// Define dynamic query parameters based on user input or other factors
const dynamicParams = {
    category: 'books',
    author: 'John Doe',
    // Additional parameters based on user input
};

// Send the GET request with the dynamic query parameters using the `params` option
axios
    .get(baseURL, { params: dynamicParams })
    .then((response) => {
        console.log(response.data);
    })
    .catch((error) => {
        console.error(error);
    });

In this example, the dynamicParams object is constructed based on user input or other dynamic factors. You can adapt the query parameters to suit your specific use case.

4. Conclusion

Sending HTTP requests with query parameters is a fundamental aspect of working with web services and APIs. Axios simplifies this process by providing convenient options for including query parameters in your requests.

Whether you need to fetch filtered data, interact with APIs, or handle dynamic user input, Axios makes it easy to create HTTP requests with query parameters and retrieve the desired data efficiently.

By mastering these techniques, you can build powerful web applications and services that leverage external data sources effectively.
