How to Filter an Array with Multiple Conditions in JavaScript?

Working with arrays is a fundamental part of JavaScript programming. Often, you'll encounter situations where you need to filter an array based on multiple conditions or criteria.

In this article, we'll explore various techniques for filtering an array with multiple conditions in JavaScript.

Understanding the filter() Method

JavaScript provides the filter() method, which is designed specifically for filtering arrays. The filter() method creates a new array with all elements that pass a given test (provided as a callback function). To filter an array with multiple conditions, you can combine these conditions in a single callback function.

Here's a basic example of how the filter() method works:

const numbers = [1, 2, 3, 4, 5, 6];

const evenNumbers = numbers.filter((number) => {
  return number % 2 === 0;
});

console.log(evenNumbers); // [2, 4, 6]

In this example, we use the filter() method to create a new array evenNumbers containing only the even numbers from the numbers array.

Filtering with Multiple Conditions

To filter an array with multiple conditions, you can write a callback function that checks all the desired conditions. You can use logical operators like && (AND) and || (OR) to combine conditions as needed.

Let's say you have an array of objects representing products, and you want to filter out products that are both in stock and have a price less than $50. Here's how you can do it:

const products = [
  { name: 'Product A', price: 40, inStock: true },
  { name: 'Product B', price: 60, inStock: true },
  { name: 'Product C', price: 30, inStock: false },
  { name: 'Product D', price: 70, inStock: false },
];

const filteredProducts = products.filter((product) => {
  return product.inStock && product.price < 50;
});

console.log(filteredProducts);

In this example, we use the filter() method with a callback function that checks if both product.inStock is true (using the && operator) and product.price is less than $50. The filteredProducts array will contain products that meet both conditions.

Using Separate Filter Steps

Another approach is to use separate filter() steps for each condition and combine the results. This approach can make your code more readable and maintainable, especially if you have complex conditions.

Here's an example:

const products = [
  { name: 'Product A', price: 40, inStock: true },
  { name: 'Product B', price: 60, inStock: true },
  { name: 'Product C', price: 30, inStock: false },
  { name: 'Product D', price: 70, inStock: false },
];

const inStockProducts = products.filter((product) => product.inStock);
const affordableProducts = products.filter((product) => product.price < 50);

const filteredProducts = inStockProducts.filter((product) => {
  return affordableProducts.includes(product);
});

console.log(filteredProducts);

In this example, we first filter the products that are in stock and those with prices less than $50 separately. Then, we use the includes() method to filter the products that are both in stock and affordable. This approach provides more flexibility and readability, especially when dealing with complex conditions.

Using the reduce() Method

While the filter() method is the most common choice for filtering arrays, you can also use the reduce() method to achieve the same result with multiple conditions. This method involves accumulating values that meet the conditions.

Here's an example:

const products = [
  { name: 'Product A', price: 40, inStock: true },
  { name: 'Product B', price: 60, inStock: true },
  { name: 'Product C', price: 30, inStock: false },
  { name: 'Product D', price: 70, inStock: false },
];

const filteredProducts = products.reduce((result, product) => {
  if (product.inStock && product.price < 50) {
    result.push(product);
  }
  return result;
}, []);

console.log(filteredProducts);

In this example, we use the reduce() method to accumulate products that meet the conditions (in stock and price less than $50) into the filteredProducts array.

Conclusion

Filtering an array with multiple conditions in JavaScript is a common and important operation in many applications. The filter() method is a powerful tool for accomplishing this task by allowing you to define custom callback functions that check all the necessary conditions. Whether you combine conditions in a single callback or use separate filter steps, JavaScript provides flexible options for filtering arrays based on your specific requirements.
