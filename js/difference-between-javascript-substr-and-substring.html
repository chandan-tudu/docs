Difference Between JavaScript substr() and substring()

JavaScript provides two similar-looking string methods: substr() and substring(), which are often used for extracting substrings from a given string.

However, there are key differences in how these methods work. In this article, we'll explore the distinctions between substr() and substring() to help you understand when to use each one in your JavaScript code.

substr(): Extract a Substring with a Starting Index and Length

The substr() method is used to extract a substring from a string based on a starting index and a specified length. Its syntax is as follows:

string.substr(startIndex, length);
  • startIndex: An integer indicating the starting position of the substring. If negative, it counts from the end of the string.
  • length (optional): The length of the substring to extract. If omitted, the method extracts characters from the startIndex to the end of the string.

Here's an example of how to use substr():

const str = "Hello, World!";
const substring = str.substr(7, 5); // Starts at index 7, extracts 5 characters
console.log(substring); // Output: "World"

In this example, we start at index 7 ("W") and extract 5 characters, resulting in "World".

substring(): Extract a Substring Between Two Indices

The substring() method is used to extract a substring from a string between two specified indices. Its syntax is as follows:

string.substring(startIndex, endIndex);
  • startIndex: An integer indicating the starting position of the substring. If negative, it is treated as 0.
  • endIndex: An integer indicating the ending position of the substring. If negative, it is treated as 0. The substring includes characters up to, but not including, the character at the endIndex.

Here's an example of how to use substring():

const str = "Hello, World!";
const substring = str.substring(7, 12); // Starts at index 7, ends at index 11
console.log(substring); // Output: "World"

In this example, we specify the start index as 7 and the end index as 12. The substring includes characters from index 7 ("W") to index 11 ("d") but does not include the character at index 12.

Key Differences

  1. Parameter Meaning:

    • substr() uses a starting index and a length to define the substring.
    • substring() uses two indices to define the substring's start and end positions.

  2. Handling Negative Indices:

    • substr() allows negative startIndex values, treating them as counting from the end of the string.
    • substring() treats negative indices as 0.

  3. End Index Inclusivity:

    • substr() is inclusive of the startIndex but stops at the character before startIndex + length.
    • substring() is inclusive of the character at the startIndex but stops just before the character at the endIndex.

Which One to Choose?

The choice between substr() and substring() depends on your specific use case:

  • Use substr() when you need to extract a substring starting from a particular index and with a specified length.

  • Use substring() when you need to extract a substring between two specific indices.

Both methods are useful for manipulating strings, and understanding their differences will help you select the appropriate method for your string manipulation tasks in JavaScript.
