How to Check if a Variable is Not NULL in JavaScript?

In JavaScript, it's common to check whether a variable has a value, especially when dealing with user input, API responses, or data from various sources.

While JavaScript doesn't have a concept of "NULL" like some other programming languages, you can check if a variable is not set, is null, or undefined using various techniques.

In this article, we'll explore different methods for checking if a variable is not null in JavaScript.

Understanding null and undefined

In JavaScript, both null and undefined represent the absence of a value, but they are used in slightly different contexts:

  • null: It is a special value in JavaScript that represents the intentional absence of any object value. Developers can assign null to a variable or object property to indicate that it has no meaningful value.

  • undefined: It is a primitive value in JavaScript that indicates a variable has been declared but has not been assigned a value. It is also used as the default return value of a function that doesn't explicitly return anything.

Method 1: Using the Equality Operator (==)

You can use the equality operator (==) to check if a variable is not null. When you compare a variable to null using ==, JavaScript considers null and undefined as equal. This means that if a variable is either null or undefined, the equality comparison will return true.

Here's an example:

let myVariable = null;

if (myVariable == null) {
  console.log("myVariable is null or undefined.");
} else {
  console.log("myVariable has a value.");
}

In this example, the variable myVariable is null, so the condition evaluates to true, and the message "myVariable is null or undefined." is logged.

Method 2: Using the Identity Operator (===)

If you want to check specifically for null and exclude undefined, you should use the identity operator (===). The identity operator checks both the value and the type of a variable, ensuring that it is exactly equal to null.

Here's an example:

let myVariable = null;

if (myVariable === null) {
  console.log("myVariable is null.");
} else {
  console.log("myVariable is not null.");
}

In this example, the condition myVariable === null evaluates to true because myVariable is null. If myVariable were undefined, the condition would evaluate to false.

Method 3: Using the != and !== Operators

To check if a variable is not null, you can also use the inequality operators (!= and !==). These operators allow you to test if a variable is not equal to null.

Here's an example with the inequality operator (!=):

let myVariable = null;

if (myVariable != null) {
  console.log("myVariable is not null.");
} else {
  console.log("myVariable is null or undefined.");
}

In this example, the condition myVariable != null evaluates to false because myVariable is null. If myVariable were undefined, the condition would evaluate to true.

Similarly, you can use the strict inequality operator (!==) to check if a variable is not null, excluding undefined.

Method 4: Using Conditional (Ternary) Operator

The conditional (ternary) operator is a concise way to check if a variable is not null and assign a value based on the condition.

Here's an example:

let myVariable = null;

let result = myVariable !== null ? "myVariable is not null." : "myVariable is null or undefined.";

console.log(result);

In this example, we use the ternary operator to assign the value of result based on whether myVariable is not null.

Method 5: Using typeof Operator

The typeof operator can be used to check if a variable is defined and not null. It returns a string indicating the data type of the variable. If the variable is not declared or null, it will return "undefined".

Here's an example:

let myVariable = null;

if (typeof myVariable !== "undefined" && myVariable !== null) {
  console.log("myVariable is not null or undefined.");
} else {
  console.log("myVariable is either null or undefined.");
}

In this example, we first use typeof to check if myVariable is not "undefined", and then we use myVariable !== null to check if it's not null.

Conclusion

Checking if a variable is not null in JavaScript is a common task when handling data and user inputs. Depending on your specific requirements and coding style, you can choose from the methods mentioned above. Whether you prefer using the equality

operator (==), the identity operator (===), the inequality operators (!= and !==), the ternary operator, or the typeof operator, JavaScript provides various options to handle null and undefined values in your code effectively.
