How to Check a Radio Button and Verify its State in JavaScript

Radio buttons are a fundamental component of HTML forms used to allow users to make a single selection from a set of options.

In JavaScript, you may encounter situations where you need to programmatically check a radio button or verify its state. In this article, we'll explore both scenarios.

Checking a Radio Button Programmatically

To check a radio button programmatically, you need to access the radio button element in the DOM and set its checked property to true. Here's how you can do it:

HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Checking a Radio Button in JavaScript</title>
</head>
<body>
    <h2>Radio Buttons</h2>
    <input type="radio" id="radioButton1" name="radioGroup" value="option1"> Option 1<br>
    <input type="radio" id="radioButton2" name="radioGroup" value="option2"> Option 2<br>
    <input type="radio" id="radioButton3" name="radioGroup" value="option3"> Option 3<br>

    <button id="checkRadioButton">Check Option 2</button>
</body>
</html>

JavaScript to Check a Radio Button

Here's the JavaScript code to check a specific radio button when a button is clicked:

// Get references to the radio button and button elements by their IDs
const radioButton2 = document.getElementById("radioButton2");
const checkRadioButtonButton = document.getElementById("checkRadioButton");

// Add a click event listener to the button
checkRadioButtonButton.addEventListener("click", () => {
    // Check the radio button
    radioButton2.checked = true;
});

In this code:

  • We use getElementById to obtain references to the radio button with the ID radioButton2 and the button with the ID checkRadioButton.
  • We add a click event listener to the button. When the button is clicked, it triggers the function that sets the checked property of the radio button to true, effectively checking it.

Verifying the State of a Radio Button

To verify whether a radio button is checked, you can access its checked property. Here's how to do it:

JavaScript to Verify Radio Button State

Here's the JavaScript code to verify the state of a radio button:

// Get references to the radio button elements by their IDs
const radioButton1 = document.getElementById("radioButton1");
const radioButton2 = document.getElementById("radioButton2");
const radioButton3 = document.getElementById("radioButton3");

// Add a click event listener to each radio button
radioButton1.addEventListener("click", () => {
    if (radioButton1.checked) {
        console.log("Option 1 is selected");
    }
});

radioButton2.addEventListener("click", () => {
    if (radioButton2.checked) {
        console.log("Option 2 is selected");
    }
});

radioButton3.addEventListener("click", () => {
    if (radioButton3.checked) {
        console.log("Option 3 is selected");
    }
});

In this code:

  • We use getElementById to obtain references to the radio buttons with IDs radioButton1, radioButton2, and radioButton3.
  • We add click event listeners to each radio button. When a radio button is clicked, it triggers the respective function.
  • Inside the event listener functions, we use an if statement to check the checked property of the radio button. If it's true, we log a message indicating that the corresponding option is selected.

Now, when you click the radio buttons, you will see messages in the console indicating which option is selected.

By following these examples, you can effectively check radio buttons programmatically and verify their states in your JavaScript-powered web applications.
